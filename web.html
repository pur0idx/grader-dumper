    การรับ input/แสดง output ปริมาณมาก ๆ

    • C++ ถ้าต้องการแสดง output จำนวนมาก ๆ ให้ใช้ printf หรือถ้าอยากใช้ cout ก็ให้ใช้ "\n" แทน endl
    • C++ ถ้า input มีจำนวนมาก ให้ใช้ scanf หรือไม่ก็เพิ่มบรรทัดนี้ไว้ตอนเริ่มต้นของ main
      • ios_base::sync_with_stdio(false); cin.tie(0);
    • Python ควรใช้ sys.stdin.readline() แทนคำสั่ง input()
    • Java ควรใช้ BufferedReader ในการรับ input ขนาดเยอะ ๆ
